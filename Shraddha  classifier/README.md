# Shraddha Kapoor Classifier

A deep learning project that implements an image classification model to recognize and classify images of Shraddha Kapoor.

## Project Overview

This project demonstrates the application of convolutional neural networks (CNNs) for celebrity image classification. The model is trained to identify images of actress Shraddha Kapoor with high accuracy using transfer learning and modern deep learning techniques.

## Contents

- **Shraddha  kapoor classifier.ipynb** - Complete Jupyter notebook containing the entire implementation pipeline
- **Shraddha_Kapoor_Classifier_Project_Report.pdf** - Detailed project report with analysis and results
- **img6.png, img7.png, img8.png** - Sample visualizations and results

## Key Features

- **Image Classification**: Binary/multi-class classification of celebrity images
- **Transfer Learning**: Leverages pre-trained deep learning models for improved accuracy
- **Data Preprocessing**: Comprehensive image processing and augmentation pipeline
- **Model Evaluation**: Detailed performance metrics and validation results

## Technologies Used

- Python
- TensorFlow / Keras
- Pandas & NumPy
- Matplotlib & Seaborn
- Jupyter Notebook

## Getting Started

### Prerequisites

```bash
pip install tensorflow keras numpy pandas matplotlib seaborn jupyter
```

### Usage

1. Open the Jupyter notebook:
```bash
jupyter notebook "Shraddha  kapoor classifier.ipynb"
```

2. Execute cells sequentially to:
   - Load and explore the dataset
   - Preprocess images
   - Train the classification model
   - Evaluate model performance
   - Generate predictions
